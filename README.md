# API Transações
### Tecnologias usadas:
+ C#.
+ MySQL.
+ Princípios SOLID.
+ Domain Driven Design(DDD).
+ Programação Orientada a Objetos.
### Como rodar o programa:
Primeira opção - Makefile:<br>
```bash
make
```
Segunda opção - Forma nativa:<br>
```bash
dotnet run --project src/CashFlow.API
```
